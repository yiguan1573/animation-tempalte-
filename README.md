# 这是一个基于vue2的动画弹幕项目，借鉴了B站和OmoFun
- 本项目为模板，没有增加网络请求，都是静态的数据和资源，方便使用
- 引用的包
  - animation.css
  - vuex
  - router
  - vue-core-video-player
  - vue-barrage
  - timers-browserify
- 该项目弹幕在全屏切换过程中会有bug
